# Color Palette

## Brand
| Key | Hex | Usage |
|-----|-----|-------|
| Primary | `#DE0F21` | Red accent, buttons, FAB |
| PrimaryDark | `#FF3B4D` | Primary in dark theme |
| Secondary | `#39485F` | Secondary accent |

## Backgrounds & Surfaces
| Light | Dark | Usage |
|-------|------|-------|
| `#FFFFFF` Background | `#121212` BackgroundDark | Page background |
| `#F7F9FC` Surface | `#1E1E1E` SurfaceDark | Cards, elevated areas |
| `#F1F3F6` SearchBarBackground | `#1E1E1E` SearchBarBackgroundDark | Search bar, unselected chips |

## Text
| Light | Dark | Usage |
|-------|------|-------|
| `#1A1A1A` TextPrimary | `#FFFFFF` TextPrimaryDark | Headings, body |
| `#6B7280` TextSecondary | `#B0B3B8` TextSecondaryDark | Subtitles, hints |

## Borders & Skeleton
| Light | Dark | Usage |
|-------|------|-------|
| `#E5E7EB` Border | `#2A2A2A` BorderDark | Card/chip borders |
| `#E5E7EB` Skeleton | `#2A2A2A` SkeletonDark | Loading placeholders |

## Status
| Key | Hex | Usage |
|-----|-----|-------|
| Error | `#DC2626` / `#EF4444` dark | Validation errors, weak password |
| Success | `#16A34A` | Strong password, success toasts |
| Warning | `#F59E0B` | Medium password, used badge |

## Listing-specific
| Key | Hex | Usage |
|-----|-----|-------|
| ConditionNewBadge | `#1A1A1A` | "New" condition badge |
| ConditionUsedBadge | `#F59E0B` | "Used" condition badge |
| PlaceholderCat0–9 | Pastel palette (light) / Deep tones (dark) | Category placeholder image backgrounds when listing has no photo |
| PlaceholderCatDefault | `#F5F5F5` / `#424242` dark | Fallback placeholder |

## Chips (selected state)
